## ğŸ—ï¸ **Lesson Structure**  

### **Day 1: Introduction to Financial Uncertainty**  
| Time | Activity | Resources |
|------|----------|-----------|
| 10 min | Icebreaker: "Money Decisions" discussion | Slide deck |
| 15 min | **Stock Price Adventure** game | Dice, worksheets |
| 30 min | Python basics: variables, loops, plotting | [notebook_1.ipynb](notebooks/) |
| 20 min | First simulation: Simple stock paths | Starter code |
| 15 min | Reflection & Kahoot! quiz | [Quiz link]() |

### **Day 2: Station Rotation - Finance Quest**  
Four interactive stations (rotate every 10-12 minutes):  

1. **Station 1: Basic Option Pricing**  
   - Learn call/put options  
   - Use simplified Black-Scholes calculator  
   - Badge: "Option Oracle"  

2. **Station 2: Jump Process Simulations**  
   - Model sudden market changes  
   - Compare smooth vs. jumpy paths  
   - Badge: "Jump Master"  

3. **Station 3: Market Completeness Puzzles**  
   - Visual matrix exercises  
   - Add/remove assets to complete markets  
   - Badge: "Market Maker"  

4. **Station 4: Monte Carlo Game**  
   - Random simulation competition  
   - Lowest error wins  
   - Badge: "Simulation Champion"  

### **Day 3: Project & Assessment**  
- **Option Pricing Project** (45 min)  
- **Peer Code Review** (20 min)  
- **Ethics Discussion:** "Fair Markets" (15 min)  
- **Exit Ticket & Reflection** (10 min)
